# Wallet Mobile

## 🚀 Tech Stack

**Client:** Vue.js, Quasar
## ⚙️ Run Locally

Clone the project

```bash
  git clone https://github.com/bitsongofficial/wallet-mobile.git
```

Go to the project directory

```bash
  cd wallet-mobile
```

Install dependencies

```bash
  yarn
```

Start the server

```bash
  yarn ios
```
or
```bash
  yarn android
```
